This project is an E-commerce system like Amazon, eBay, craiglist is one that
allows users to buy and sell products - Books and Musics (B&M)

This project is build with PHP and uses MySQL as its database back-end.

This project includes the following steps http://sdrv.ms/15cAcoP:
  Step 1: Brainstorm what the user of the website to buy and sell
  Step 2: Develop complete E-R diagram
  Step 3: Implement the design in database
  Step 4: Implement all the other components of the website
